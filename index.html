Comment créer un site web en html ? (cette méthode marche avec chrome OS):

  1. Pour créer en local(seulement sur votre machine):

    Créez un fichier texte puis écrivez votre code et remplacez ".txt" par ".html". Lancez le fichier et vous voilà sur votre page web!

  2. Pour créer en publique (sur n'importe quelle machine) :

    Le problème de la methode précédente est que vous n'y avez accès qu'en local: pour que quelqu'un qui a une autre machine le voit on peut passer par github, si vous n'avez pas encore de compte il faut créer un compte github . Après cela vous puvez aller voir cet article car c'est la suite.
